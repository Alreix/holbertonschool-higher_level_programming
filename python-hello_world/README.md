# Python - Hello, World

This project is the first step into high-level programming with Python at Holberton School.  
It focuses on using the Python interpreter, printing text and variables, and working with basic string operations.

---

## Learning Objectives

By the end of this project, you should be able to explain to anyone, without the help of Google:

- How to use the Python interpreter
- How to print text and variables using `print`
- How to use strings in Python
- What indexing and slicing are in Python
- What the official Python coding style is and how to check your code with `pycodestyle`

---

- Editors: `vi`, `vim`, `emacs`
