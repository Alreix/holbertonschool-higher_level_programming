## Project Structure

- **python-hello_world**  
  Introduction to Python syntax and basic operations.

- **python-if_else_loops_functions**  
  Control flow, functions, and logical operations.

- **python-import_modules**  
  Using modules, importing functions, and working with scripts.

- **python-data_structures**  
  Lists, tuples, and fundamental data manipulation.

- **python-more_data_structures**  
  Sets, dictionaries, lambda, map, filter, reduce.

- **python-exceptions**  
  Handling errors and raising exceptions.

- **python-test_driven_development**  
  Writing tests using docstring and unittest.

- **python-classes**  
  Object-Oriented Programming, classes, instances, attributes, and methods.
