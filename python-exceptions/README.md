# Python - Exceptions

This directory contains a set of Python exercises focused on exception handling and error management.  
The goal of the project is to learn how to use `try/except`, raise exceptions, handle multiple error cases, and ensure safe execution of functions.

---

## Project Structure

- `0-safe_print_list.py`  
  Function that prints elements of a list using `try/except`.

- `1-safe_print_integer.py`  
  Function that prints an integer and returns `True` or `False` based on success.

- `2-safe_print_list_integers.py`  
  Function that prints integers from a list while safely handling non-integer values.

- `3-safe_print_division.py`  
  Function that performs a division and prints the result using exception handling.

- `4-list_division.py`  
  Function that divides element-by-element two lists while managing different error cases.

- `5-raise_exception.py`  
  Function that raises a standard exception.

- `6-raise_exception_msg.py`  
  Function that raises an exception with a custom message.

---

## Requirements

- Python 3.8.5 (Ubuntu 20.04 LTS)
- Code must follow **pycodestyle** (version 2.7.\*)
- All modules and functions must be documented using docstrings
- No external libraries are allowed
